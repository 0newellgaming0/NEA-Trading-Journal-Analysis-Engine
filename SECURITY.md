# Security Policy

## Scope
This project handles financial analysis logic and should not be exposed to sensitive credentials.

## Supported Versions
We actively support the latest version of this project.

## Safe Practices
- No API keys committed
- No broker credentials stored
- No live trading execution in core repo

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly:

- Do NOT open a public issue
- Contact the maintainer directly via private message or email

## Response Timeline
We aim to respond within 72 hours and provide a resolution or mitigation plan as soon as possible.

Thank you for helping keep this project secure.
