📘 Institutional Marubozu Analysis Engine — README
Overview

This module implements a multi-layer institutional candlestick analysis system centered around the Marubozu candlestick pattern, enriched with:

Trend structure validation (EMA/SMA alignment)
Liquidity sweep detection (Wyckoff-style behavior)
Volume confirmation (RVOL + spikes + institutional state)
Market structure proximity (support/resistance)
Regime classification (active / contextual / no edge)
Automated trade planning (entry, stop, targets)

It is designed for algorithmic trading workflows where candlestick signals must be validated through institutional confluence, not isolated pattern detection.

🧠 Core Design Philosophy

Unlike traditional candlestick detectors, this module treats price action as:

A liquidity-driven auction process, not a pattern catalog

A Marubozu candle is only meaningful when confirmed by:

Trend alignment
Volume expansion
Liquidity context
Structural positioning
Follow-through or rejection behavior
📦 Dependencies
import numpy as np

Case B (Ticker-Suffixed Columns)

The module automatically detects and normalizes structure.

🕯 Marubozu Detection Engine
7. detect_marubozu(data)
Definition

A Marubozu is defined as:

Large real body (≥ 75% of candle range)
Minimal wicks (≤ 10%)
Directional dominance (bullish or bearish)
Output Structure
{
  "detected": bool,
  "type": "Bullish" | "Bearish",
  "strength": "Institutional | Strong | Standard | Weak",
  "range": float,
  "body_ratio": float,
  "high": float,
  "low": float,
  "open": float,
  "close": float
}
Strength Classification
Body Ratio	Strength
≥ 0.90	Institutional
≥ 0.80	Strong
≥ 0.75	Standard
< 0.75	Weak (rejected)

🔁 Confirmation Engine
10. confirm_marubozu_today(today, y_pin)
Purpose

Determines whether yesterday’s Marubozu is:

State	Meaning
CONFIRMED	Follow-through continuation
FAILED	Breakdown against direction
LIQUIDITY_REJECTION	Wick-through + rejection
PENDING	No decisive outcome
Logic Core
Bullish Marubozu:
Close > high → CONFIRMED
Close < low → FAILED
Wick above high + close back inside → REJECTION
Bearish Marubozu:
Reverse logic
