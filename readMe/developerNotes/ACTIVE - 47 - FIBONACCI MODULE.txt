📘 Fibonacci Pullback Module (Swing-Based Institutional Version)
Technical Strategy Plugin — Resolver Compatible Event Engine
1. Overview

The Fibonacci Pullback Module is a structured institutional swing-retracement system designed to detect trend-continuation pullbacks using Fibonacci zones derived from dynamic swing structure.

Unlike static Fibonacci tools used in retail charting, this module builds a real-time swing engine + trend classifier + liquidity-aware retracement model.

Its purpose is not just to “draw fib levels”, but to:

Identify active institutional trend structure
Detect first meaningful retracement into value zones
Confirm continuation or failure via structural invalidation
Convert fib behavior into a trade-state machine
2. Core Philosophy

This module is built around three institutional principles:

2.1 Market as Swing Auction

Price is assumed to move in:

Expansion → Distribution → Retracement → Re-accumulation → Continuation

Fib levels represent:

Probability zones of re-entry, not static support/resistance
2.2 Trend Precedes Fibonacci

Fib is only valid if:

Trend structure exists (HH/HL or LL/LH)
Swing structure supports directional bias

No trend = no valid fib signal.

2.3 Pullbacks Are Liquidity Events

Pullbacks are interpreted as:

Stop hunts
Repricing zones
Institutional re-entry zones

The module explicitly seeks:

“First meaningful retracement into institutional discount/premium zones”

3. System Architecture

The module is divided into five functional layers:

Layer 1 — Trend Engine
get_trend(df, i)

Detects:

BULLISH → Higher High + Higher Low structure
BEARISH → Lower Low + Lower High structure
RANGE → No directional structure

Purpose:
Establish directional bias BEFORE fib is applied.

Layer 2 — Swing Engine (Dynamic)
get_latest_swing(df, i)

This is NOT a static high/low system.

It dynamically adjusts swing structure based on:

Lookback range (25 bars)
Recent micro-structure (5 bars)
Breakout behavior
Modes:
EXPANSION_UP
Price breaks above swing high
Uses recent highs for refinement
EXPANSION_DOWN
Price breaks below swing low
Uses recent lows for refinement
RANGE
Normal swing range context

Purpose:
Builds institutional-grade swing anchoring instead of retail static fibs.

Layer 3 — Fibonacci Engine
get_fib_levels()

Standard retracement grid:

0.0 → Swing High
0.236 → Early pullback
0.382 → First institutional dip zone
0.5 → Equilibrium zone
0.618 → Primary institutional entry zone
0.786 → Deep retracement / stress zone
1.0 → Swing Low

Key Level:

0.618 = primary continuation trigger zone

4. Detection Logic (Core Signal Engine)
detect_fib_pullback()
4.1 Seed Condition (MOST IMPORTANT)

A Fibonacci pullback seed is triggered when:

Bullish Setup:
Trend = BULLISH
Price closes BELOW 0.382 retracement
Bearish Setup:
Trend = BEARISH
Price closes ABOVE 0.618 retracement
Interpretation:

This represents:

First meaningful violation of equilibrium
Early liquidity pullback into fib zone
Potential institutional re-entry trigger
5. Entry Logic (Activation Phase)
detect_fib_entry()
Entry Trigger Rule:
Bullish: close ≤ 0.618 level
Bearish: close ≥ 0.618 level
Meaning:

This is NOT a reversal signal.

It is:

“Price has reached institutional discount/premium zone inside trend”

6. Confirmation Logic (Continuation Validation)
detect_fib_confirmation()
Bullish Confirmation:
Close > swing high
Bearish Confirmation:
Close < swing low
Meaning:

This is:

Breakout continuation of prior swing direction
Full resumption of trend expansion
7. Failure Logic (Critical Structural Invalidations)
detect_fib_failure()
Bullish Failure:
Close < swing_low
Bearish Failure:
Close > swing_high
Meaning:

This represents:

Trend invalidation
Structural breakdown of swing logic
Institutional exit / reversal environment
8. Event State Machine

The system operates as a 3-stage progression model:

🟡 SEED

Initial detection phase

Trend exists
Fib pullback begins
No entry yet confirmed
🔵 ENTRY_TRIGGER

Price enters institutional fib zone

0.618 level engaged
Retest or reaction expected
🟢 CONFIRMED

Trend continuation validated

Break of swing high/low
Full directional resumption
🔴 FAILED

Structure invalidated

Swing low/high breached
Trend no longer valid
