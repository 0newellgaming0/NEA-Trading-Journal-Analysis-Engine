5. Point & Figure Construction Logic
5.1 Core Concept

P&F charts are built using:

Box size (box_size)
Reversal threshold (reversal)
5.2 Column Types
"X" → Rising demand column
"O" → Falling supply column
5.3 Reversal Rules

A reversal occurs when price moves:

Against current direction
By at least reversal × box_size
6. Pattern Detection Logic
6.1 Springs (Bullish Trap Reversal)

Definition:

Price breaks below support (O column)
Then quickly reverses upward (X column)

Interpretation:

Institutional accumulation signal
Liquidity sweep
6.2 Upthrusts (Bearish Trap Reversal)

Definition:

Price breaks above resistance (X column)
Then fails and reverses downward (O column)

Interpretation:

Distribution phase
False breakout trap
6.3 Triple Top / Bottom
Triple Top: repeated resistance failure breakout structure
Triple Bottom: repeated support breakdown structure

These indicate:

Exhaustion of supply/demand zones
7. Volume Intelligence System
7.1 Volume States
EXPANDING → institutional participation
CONTRACTING → consolidation / absence of interest
NORMAL → equilibrium
7.2 Volume Trend Slope

Computed using polynomial regression:

poly_slope(volume_array)

Used to detect:

Accumulation (declining volume, stable price)
Distribution (rising volume, weak price action)
8. Wyckoff Phase Engine
8.1 Phase Classification
Condition	Phase
Vol ↓ + Price ↑	Accumulation
Vol ↓ + Price ↓	Distribution
Vol ↑ + Price ↑	Markup
Vol ↑ + Price ↓	Markdown
8.2 Compression Metric
compression = (High - Low) / Close

Used to measure:

volatility contraction
breakout readiness
9. Target Projection System
9.1 Horizontal Target
resistance + width
9.2 Vertical Target
resistance + (width × 1.5)
9.3 Count Target
resistance + (congestion columns × box_size)
10. Signal Engine Logic
10.1 BREAKOUT
Price breaks above resistance in X column
Strength = 0.9
10.2 BREAKDOWN
Price breaks below support in O column
Strength = 0.9
10.3 REVERSAL SIGNALS
X after O → bullish reversal
O after X → bearish reversal
10.4 Volume Confirmation

If volume confirms:

Strength increases by +0.1 (max 1.0)
