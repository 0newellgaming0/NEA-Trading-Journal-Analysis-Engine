📘 Three Inside Pattern Detection Module
Overview

The Three Inside Pattern Detection Module is a quantitative candlestick analysis engine designed to identify, classify, and generate trade plans for the Three Inside Up / Three Inside Down candlestick formation.

This module is built for algorithmic trading systems that require:

Institutional-grade candlestick validation
Multi-stage pattern detection (Forming → Pending → Confirmed)
Automated trade plan generation
Robust OHLC normalization across ticker formats
Safe numeric extraction from Pandas structures

It is optimized for noisy real-world market data where OHLC columns may be inconsistent or multi-ticker formatted.

📌 Core Features
✔ Multi-Stage Pattern Recognition

The module identifies the Three Inside structure in three phases:

FORMING → early harami structure detected
PENDING_CONFIRMATION → structure valid but breakout not confirmed
CONFIRMED → breakout candle validates directional expansion
✔ Institutional Candle Validation

The system filters weak or invalid signals using:

Body strength thresholds
Trend direction validation (Candle 1 filter)
True harami containment logic
Breakout expansion requirement
✔ Automated Trade Planning

Once confirmed, the module generates:

Entry triggers (long/short)
Stop-loss levels
Target projections (1R and 2R extensions)
Directional bias (bullish / bearish)
✔ Robust Data Handling

Supports messy or multi-ticker datasets:

Auto ticker detection from column suffixes
Safe OHLC extraction from nested Pandas structures
Missing column fallback handling
Numeric coercion and cleanup
📂 Module Architecture
1. Data Preprocessing Layer
normalize_ohlcv_columns(df, ticker)
enforce_schema(df)
f(x) safe scalar extractor
2. Candle Utility Layer
body(c) → candle body size
rng(c) → candle range (safe division protection)
is_bull(c) / is_bear(c)
is_inside(c1, c2) → containment logic
3. Pattern Logic Layer
Candle 1 Trend Filter

Validates whether the initial candle has institutional strength:

Requires body ratio > 0.45
Determines bullish or bearish bias
True Harami Validation

Ensures Candle 2 is fully contained inside Candle 1 body:

Strict body containment (no wick-based noise)
Center Candle Classifier

Classifies Candle 2:

Type	Condition
Doji	body ≤ 10% of range
Balanced	body ≤ 30% + wick symmetry
Weak	body ≤ 40%
Invalid	otherwise rejected
Breakout Validation

Candle 3 must:

Break above Candle 1 high (bullish)
OR break below Candle 1 low (bearish)
AND show expansion (>60% of Candle 1 range)

⚙️ Core Design Philosophy
1. Institutional Filtering First

Weak signals are eliminated early via:

Body strength thresholds
Trend validation
Range expansion confirmation
2. Multi-Stage State Machine

The system avoids binary classification:

FORMING → structure awareness
PENDING → breakout anticipation
CONFIRMED → execution-ready signal
3. Expansion-Based Confirmation

Breakouts are only valid if:

Price expansion confirms participation (not noise)

4. Body-Only Structural Integrity

Harami detection is based strictly on:

Candle body containment (not wicks)
