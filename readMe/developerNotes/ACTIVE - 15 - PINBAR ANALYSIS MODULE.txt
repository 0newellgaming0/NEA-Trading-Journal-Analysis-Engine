OVERVIEW

This module implements a full institutional-grade Pin Bar detection and trade decision engine built on OHLCV market data.

It is designed to:

Detect bullish and bearish pin bars
Classify rejection strength
Track multi-day confirmation behavior
Identify liquidity sweeps (Wyckoff logic)
Evaluate trend, structure, and volume alignment
Generate structured trade plans
Output journal-ready institutional analysis reports

The system is intended for systematic trading workflows, journaling systems, and algorithmic decision support engines.

CORE DESIGN PHILOSOPHY

This module treats price as:

“A liquidity auction driven by institutional participation, rejection, and acceptance.”

Key principles:

Wick = liquidity search
Close = acceptance or rejection
Volume = institutional intent
Structure = context validation
Multi-candle behavior = confirmation logic
INPUT REQUIREMENTS

All functions expect a Pandas DataFrame with OHLCV data:

Required columns:

Open
High
Low
Close
Volume

The module can also auto-normalize ticked datasets (e.g., close_AAPL, open_TSLA).

CORE UTILITIES
1. Safe Scalar Extractor
f(x)

Safely extracts scalar float values from:

Pandas Series
NaN values
raw floats

Ensures system stability across mixed data types.

2. OHLCV NORMALIZATION
normalize_ohlcv_columns(df, ticker=None)

Converts ticker-based columns into standard format:

open_<ticker> → Open
high_<ticker> → High
low_<ticker> → Low
close_<ticker> → Close
volume_<ticker> → Volume

If ticker is not provided, it attempts auto-detection.

enforce_schema(df)

Ensures:

numeric conversion
removal of invalid OHLC rows
clean analytical dataset
VOLUME ANALYSIS ENGINE
Relative Volume (RVOL)
rvol(volume, period=20)

Measures current volume vs rolling average.

Volume Spike Detection
detect_volume_spike(volume, threshold=1.5)

Identifies abnormal institutional activity.

Institutional Accumulation State
institutional_accumulation_state(close, high, low, volume)

Classifies market state into:

Accumulation (low compression + high volume)
Distribution (high compression + high volume)
Neutral

This is a simplified Wyckoff-style regime classifier.
