📊 Institutional Star Pattern Analysis Module (README)
Overview

This module implements a multi-layer institutional candlestick intelligence engine focused on identifying, validating, and trading Morning Star and Evening Star reversal patterns within a broader market context.

It combines:

Candlestick pattern recognition (Star formations)
Institutional volume behavior (RVOL, spikes, accumulation)
Market structure analysis (support/resistance)
Liquidity sweep detection (Wyckoff-style logic)
Trend filtering (EMA-based regime detection)
Automated trade planning
Journal-ready output formatting

The system is designed for algorithmic discretionary trading support, not standalone execution.

🧠 Core Philosophy

This module is built on the principle that:

Price patterns are only valid when confirmed by structure, volume, and institutional behavior.

Therefore, every signal is filtered through:

Trend alignment (EMA 8 / EMA 21 / SMA 50)
Liquidity context (sweeps, springs, upthrusts)
Volume confirmation (RVOL + spikes + expansion candles)
Market structure (support/resistance proximity)

1. Safe Data Handling
Robust scalar extraction via f(x)

Ensures safe float conversion across all indicators
2. OHLCV Normalization Layer

open_ticker

Open, High, Low, Close, Volume

✔ Ensures compatibility across multi-ticker datasets

3. Volume Intelligence Engine
Relative Volume (RVOL)
rvol(volume, period=20)

Measures:

Current volume vs rolling average volume

Volume Spike Detection
detect_volume_spike(volume, threshold=1.5)

Flags institutional participation when:

Volume > 1.5x average
Institutional Accumulation State
institutional_accumulation_state(...)

Classifies market behavior into:

State	Meaning
Accumulation	Low compression + high volume
Distribution	High compression + high volume
Neutral	No institutional footprint
4. Star Candle Classifier ⭐
classify_star_center_candle(candle)

Classifies center candle in 3-candle star patterns:

Type	Meaning
Doji	Strong indecision
Spinning Top	Balanced uncertainty
Small Body	Weak directional pressure
Directional Candle	Invalid for star pattern
5. Star Pattern Detection Engine
detect_star_pattern(df)

Detects:

🌅 Morning Star (Bullish Reversal)

Conditions:

First candle bearish
Middle candle indecision
Third candle closes above midpoint of first candle
🌇 Evening Star (Bearish Reversal)

Conditions:

First candle bullish
Middle candle indecision
Third candle closes below midpoint of first candle
Output Example
{
  "detected": True,
  "type": "MorningStar",
  "strength": "VALID",
  "high": ...,
  "low": ...,
  "close": ...
}
