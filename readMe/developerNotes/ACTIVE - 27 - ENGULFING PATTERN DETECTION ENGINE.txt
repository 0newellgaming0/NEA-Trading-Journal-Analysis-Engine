📌 ENGULFING PATTERN ANALYSIS ENGINE (INSTITUTIONAL GRADE)
🧠 Overview

This module implements an institutional-grade candlestick analysis engine focused on:

Engulfing pattern detection (bullish & bearish)
Multi-candle confirmation logic
Liquidity-based validation (sweeps, rejection, acceptance)
Volume + expansion confirmation
Trend alignment filtering
Trade planning + journaling output

It is designed for systematic trading decision automation, not visual pattern recognition alone.

⚙️ Core Philosophy

Unlike basic candlestick scanners, this engine does NOT rely only on:

Body engulfing alone
Single-candle signals
Static pattern definitions

Instead, it evaluates:

✔ Market Context First
Trend direction (EMA 8 / EMA 21 / SMA 50)
Structure location (support/resistance proximity)
Liquidity behavior (sweeps & traps)
Volume participation (RVOL + spikes + expansion)
✔ Then Signal Validation
True body engulfing (with filtering)
Strength scaling (Standard / Strong / Institutional)
Multi-candle confirmation (yesterday vs today logic)
✔ Then Execution State
Pending → Confirmed → Failed → Rejection → Break attempt
