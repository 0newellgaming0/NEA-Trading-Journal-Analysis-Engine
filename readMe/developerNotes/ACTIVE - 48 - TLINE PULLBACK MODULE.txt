📘 Bull / Bear Pullback Module (T-Line Parity Fixed)
EMA8 Institutional Pullback Continuation Engine
1. Overview

The Bull / Bear Pullback Module is a structured trend-continuation system built around the EMA8 (T-Line) framework, institutional trend filters, and controlled pullback behavior.

Its primary purpose is to detect:

“High-probability continuation setups that occur when price temporarily displaces away from the EMA8 inside a confirmed trend regime.”

Unlike generic EMA crossover systems, this module is designed as a state-driven pullback engine, not a signal indicator.

2. Core Philosophy

This module is built on three institutional assumptions:

2.1 Trend Structure Dominates Everything

Pullbacks are only valid if:

EMA8 is aligned with SMA50
SMA50 is aligned with SMA200
Market is in a directional regime

No alignment = no valid pullback signal.

2.2 Pullbacks Are Displacement Events

A pullback is not a reversal.

It is:

Temporary deviation from EMA8
Liquidity capture against trend direction
Preparation phase for continuation expansion
2.3 EMA8 is the Execution Trigger Line

The EMA8 (T-Line) represents:

Institutional short-term control line
Dynamic equilibrium between momentum and retracement

Price interaction with EMA8 defines state transitions.

3. System Architecture

The module is structured into five logical layers:

Layer 1 — Trend Engine
bullish_trend = ema8 > sma50 and sma50 >= sma200
bearish_trend = ema8 < sma50 and sma50 <= sma200
Trend Conditions:
Bullish Regime
EMA8 above SMA50
SMA50 above SMA200
Bearish Regime
EMA8 below SMA50
SMA50 below SMA200
Layer 2 — Primary Levels Engine
get_primary_levels()

Computes short-term structure:

Support = lowest low in lookback window
Resistance = highest high in lookback window
Range = structural volatility
Midpoint = equilibrium zone
Purpose:
Contextualizes pullback depth
Defines structural bounds for invalidation
Layer 3 — Pullback Seed Detection
detect_pullback()
SEED Condition (Critical Concept)

A seed is triggered when:

Bullish Trend:
Price is BELOW EMA8
Candle closes bearish (close < open)
Bearish Trend:
Price is ABOVE EMA8
Candle closes bullish (close > open)
Interpretation:

This represents:

First meaningful displacement against trend
Early retracement phase
Momentum exhaustion at EMA8 boundary

Not a reversal — a continuation setup in formation

4. Entry Logic (Continuation Displacement)
detect_pullback_entry()
Entry Rule:
Bullish continuation → close < EMA8
Bearish continuation → close > EMA8
Meaning:

“Price has extended into displacement territory relative to EMA8 control line”

5. Confirmation Logic (Trend Resumption)
detect_pullback_confirmation()

Bearish Confirmation:

6. Failure Logic (Structural Break)
detect_pullback_failure()
Bullish Failure:

Bearish Failure:
Close > seed high
Meaning:

This indicates:

Trend structure has been violated
Pullback became reversal
Institutional control lost
