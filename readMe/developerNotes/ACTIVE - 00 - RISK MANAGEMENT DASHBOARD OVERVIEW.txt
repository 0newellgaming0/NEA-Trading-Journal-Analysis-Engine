OVERVIEW

The NEA28V1 Risk Management Dashboard is a comprehensive desktop
trading risk management, journaling, and institutional analysis tool
designed for structured trade planning, execution tracking, and post-trade
review.

It integrates deterministic risk modeling, Fibonacci-based ladder entries,
and a full trade journal system with editable records, AI analysis storage,
and multi-tab trade review workflows.

This system has evolved into a full trade lifecycle management platform:

Pre-trade risk modeling
Structured entry planning
Trade execution journaling
AI-driven institutional analysis storage
Post-trade editing and updates
Historical execution tracking (Webull import)
Full CSV persistence layer

INSTALLATION

Ensure Python 3.8+ is installed
Tkinter is required (bundled with most Python installs)

Linux:
sudo apt-get install python3-tk

Place the following in the same directory:

No external dependencies required.

CALCULATIONS & FORMULAS

Risk Model:

Risk ($) = Account Size × Risk %

Ladder Range:

Range = Last High − Low

Fibonacci Ladder Levels:

Wave 1 = Low + (Range × 0.31298)
Wave 2 = Low + (Range × 0.45)
Shakeout = Low + (Range × -0.238)

Position Sizing:

Shares = (Risk $ × Allocation) ÷ (Entry − Stop)

Allocations:

12.5%
50%
25%
12.5%

Buy Now Model:

Shares = Risk $ ÷ (BuyNow − Stop)

Risk/Reward:

Target = Entry + (Entry − Stop)
