🧠 1. MODULE OVERVIEW

The Candle Over Candle Module is a volatility expansion and liquidity detection system designed to identify:

Full Outside Bar Expansion Events
Structural range engulfments
Momentum-driven institutional participation
Neutral compression breaks transitioning into directional trends
Failed or successful breakout confirmations

It operates as a two-phase event engine:

Detection Layer (Pure Pattern Recognition)
State Machine Layer (Progressive Validation Engine)
⚙️ 2. CORE DESIGN PHILOSOPHY

This module is built around the concept:

“Price reveals intent when it fully engulfs prior market range.”

A Candle Over Candle event is not just a large candle — it represents:

Liquidity sweep across both sides of prior structure
Stop-loss harvesting above and below prior candle range
Institutional participation in volatility expansion
Transition from compression → expansion regime
📊 3. STRUCTURAL DEFINITION

A valid Candle Over Candle event requires:

✔ RANGE ENGULFMENT (PRIMARY CONDITION)

Current candle must satisfy:

High > Previous High
Low < Previous Low

This defines a true outside bar structure

✔ BODY CONTEXT (DIRECTIONAL INTENT)

We evaluate:

Current candle body direction
Previous candle direction
Internal body expansion vs previous body
✔ EXPANSION STRENGTH

Measured using:

body_ratio = body / range

Interpretation:

0.60 → EXPANSION

≤ 0.60 → MODERATE
📈 4. DETECTION TYPES
🟡 NEUTRAL (COMPRESSION BREAK)

Occurs when:

Full range engulfment exists
No directional dominance yet confirmed

Represents:

Liquidity sweep
Breakout uncertainty
Early expansion phase
🟢 BULLISH CANDLE OVER CANDLE

Triggered when:

Current candle is bullish
Previous structure is bearish OR body engulfment occurs
Full range expansion is present

Represents:

Buyer dominance after liquidity sweep
Structural reclaim of downside liquidity
Early trend continuation signal
🔴 BEARISH CANDLE OVER CANDLE

Triggered when:

Current candle is bearish
Previous structure is bullish OR body engulfment occurs
Full range breakdown occurs

Represents:

Seller dominance after upside liquidity sweep
Distribution after failed breakout
Downside continuation signal
🧬 5. SEED vs FULL EVENT MODEL

Although this module does not explicitly split seed/full functions like others, it behaves in two phases:

🌱 SEED STATE (Implicit)

Initial detection occurs when:

Outside bar forms
Direction is still ambiguous OR developing
No confirmation beyond extremes yet
🚀 CONFIRMED STATE

Occurs when:

Price breaks and closes beyond expansion boundary
Directional intent is validated
🔁 6. EVENT STATE MACHINE

The module transitions through:

🟡 PENDING

Initial detected state

🟢 CONFIRMED

Breakout or breakdown validated

🔴 FAILED

Price rejects expansion and invalidates structure

STATE TRANSITIONS
NEUTRAL → CONFIRMED
Break above high → Bullish
Break below low → Bearish
BULLISH / BEARISH → FAILED
Close against expansion direction
