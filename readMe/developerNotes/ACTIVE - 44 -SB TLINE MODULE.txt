🧠 CORE PHILOSOPHY

The T-Line system is based on one fundamental principle:

Price relative to the 8 EMA defines trend control.

Above EMA8 → Buyers control trend (bullish bias)
Below EMA8 → Sellers control trend (bearish bias)

When combined with structural validation (high/low extremes), this creates a continuation framework rather than a predictive indicator system.
