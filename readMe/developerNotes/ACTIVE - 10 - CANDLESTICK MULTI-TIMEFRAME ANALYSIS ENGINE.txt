This module is a complete institutional-grade candlestick analysis engine designed to:

Normalize and clean OHLCV market data
Detect single, two-candle, three-candle, and advanced candlestick patterns
Classify patterns with interpretation + actionable trading logic
Confirm patterns using volume + trend + follow-through price action
Generate institutional trade setups (entry, stop, target)
Evaluate pattern strength (Institutional Grade scoring system)
Perform multi-timeframe alignment analysis (15M, 1H, DAILY)
Produce journal-ready structured outputs

It is built to function as a decision support layer for systematic trading systems, not just a pattern detector.

5. CANDLE STRUCTURE ENGINE
==========================================================
candle_structure(open, high, low, close)

Computes structural candle properties:

Outputs:
Body → absolute candle body size
Range → high-low range
BodyRatio → body/range ratio (strength indicator)
UpperShadow → wick above body
LowerShadow → wick below body
Purpose:

This is the foundation for:

Doji detection
Reversal classification
Volatility analysis
