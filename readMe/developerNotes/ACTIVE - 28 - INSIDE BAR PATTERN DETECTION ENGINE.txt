# 🧠 Inside Bar Institutional Analysis Engine

A high-precision **candlestick pattern detection and liquidity-based trading engine** designed to identify, classify, and operationalize **Inside Bar setups** using institutional price-action logic.

This module goes beyond simple pattern recognition — it models Inside Bars as:

> **Liquidity compression → Expansion OR Liquidity sweep → Reversal**

It is designed for systematic trading workflows, journal automation, and multi-timeframe integration.

---

# 📌 Core Concept

An **Inside Bar** is defined as:

- A “mother candle” followed by a candle completely contained within its high and low.
- Represents **market compression** (volatility contraction).
- Precedes:
  - Expansion (trend continuation), OR
  - Liquidity sweep (false breakout → reversal)

This module formalizes that behavior into:

- Pattern detection
- Breakout classification
- Event-state tracking
- Trade planning engine
- Journal-ready output

## 2. OHLCV Normalization Engine

### `normalize_ohlcv_columns(df, ticker=None)`

Converts multi-ticker or prefixed datasets into a standardized OHLCV format.

### Input formats supported:

open_AAPL, high_AAPL, low_AAPL, close_AAPL


### Output schema:

### Key features:

## 4. Inside Bar Detection Engine

### `detect_inside_bar(df)`

### Logic:
A candle is an **inside bar** if:


Inside High ≤ Mother High
AND
Inside Low ≥ Mother Low


### Outputs:
```json
{
  "detected": true,
  "mother_high": 100.0,
  "mother_low": 95.0,
  "inside_high": 99.0,
  "inside_low": 96.0,
  "compression_ratio": 0.42,
  "type": "Inside Bar"
}
Key Metric:
Compression Ratio
Measures inside candle body relative to mother candle
Lower = stronger consolidation
5. Breakout Classification Engine
classify_inside_breakout(df, inside)

Classifies post-inside-bar behavior into institutional states:

🟢 Bullish Confirmation
Close > Mother High

→ BULLISH_BREAKOUT_CONFIRMED

🔴 Bearish Confirmation
Close < Mother Low

→ BEARISH_BREAKOUT_CONFIRMED

⚠️ Liquidity Trap (False Breaks)
Bull Trap:
High > Mother High
BUT Close ≤ Mother High

→ BULLISH_FALSE_BREAK

Bear Trap:
Low < Mother Low
BUT Close ≥ Mother Low

→ BEARISH_FALSE_BREAK

🟡 Default State:

→ PENDING
