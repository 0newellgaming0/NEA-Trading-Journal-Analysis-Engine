# Code of Conduct

## Our Pledge
We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior
- Use respectful and professional language
- Accept constructive feedback
- Focus on what is best for the project
- Show empathy toward other contributors

## Unacceptable Behavior
- Harassment, insults, or discriminatory language
- Personal attacks
- Public or private harassment
- Sharing private information without permission

## Enforcement
Project maintainers may remove, edit, or reject contributions that do not align with this Code of Conduct.

## Reporting
Report unacceptable behavior by opening a SECURITY or SUPPORT discussion or contacting the maintainer directly.
